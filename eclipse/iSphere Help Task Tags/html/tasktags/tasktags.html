Task Tags

Task Tags are used as a reminder for any work or action that needs to be done by a programmer on a given source code. Basically tasks are added to a source member as comments composed of a string (= tag) followed by a short description. Well known task tags are:
  • TODO
  • FIXME
Examples:

RPG: C* TODO Change to /free
RPGLE:// TODO Enhance comments
CLP: /* FIXME Add error handler */

The LPEX task tags subproject introduces tags taks to the IBM LPEX editor. In order to keep complexity and confusion low the project shares the Java Task Tags preferences page to maintain the tags.

The Java task tags preferences page is accessible from Windows | Preferences | Java | Compiler | Task Tags:

Notice: The 'Case sensitive task tag names' checkbox is ignored for LPEX task tags.
