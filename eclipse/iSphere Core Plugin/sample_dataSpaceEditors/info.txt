Sample Data Space Editors
-------------------------

The content of this folder needs to be copied to the
following location, to see the sample editors in the
"iSphere Data Space Editor Designer":

  [workspace]\.metadata\.plugins\biz.isphere.core\dataSpaceEditors

The editors match with the following supplied test
data areas like this:

  Editor              Object
  --------------      ---------------------------
  TSTDTAARA1 - 1      TSTDTAARA1 (*DTAARA *CHAR)
  TSTDTAARA2          TSTDTAARA2 (*DTAARA *DEC)
  TSTDTAARA3          TSTDTAARA3 (*DTAARA *LGL)
  TSTUSRSPC1 - 1      TSTDTAARA1 (*DTAARA *CHAR)
                      TSTUSRSPC1 (*USRSPC)

The editor "TSTUSRSPC1 - 1" has both objects assigned.
