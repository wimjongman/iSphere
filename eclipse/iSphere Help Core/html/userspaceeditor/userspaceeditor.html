User Space Editor

The User Space Editor is used to edit user spaces.

To edit a user space:

  • Select a user space in the RSE tree
  • In the context menu (right click) select "iSphere User Space Editor"

After you have made your edits, close the editor and you will be prompted to save the changes:

You can use the standard Eclipse search to search for hexadecimal or string values:

You can use one of the following notations to search for hexadecimal values:
  • 0xE8FA32
  • x'E8FA32'
  • xE8FA32

Note: Today options 'Whole word' and 'Selected lines' are not supported.
