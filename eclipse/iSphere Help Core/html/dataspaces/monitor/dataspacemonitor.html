Data Space Monitor

The Data Space Monitor is used for monitoring the data of your data space objects. Prior you can use the monitor for a specific object, you need to go to the Data Space Editor Designer and create an editor for it.

Once you have one or more editors for your space object, you can start the monitor from the context menu of the RSE tree. In case you have more than one editor for the selected object, you need to select the desired editor from the dialog that showed up:

Now the monitor is opened and the content of your data space shows up as defined by the editor.
