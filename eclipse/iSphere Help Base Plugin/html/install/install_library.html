Install the iSphere Library

Although it is recommend to use the integrated upload feature to install the iSphere library on the host system, you can do it by hand, of course. Both processes are described below.

Using the Integrated Upload Feature

  • Start WDSCi 7.0 / RDi 8.0+.
  • Click on "Windows / Preferences".
  • Expand item "iSphere".
  • Click "Library".
  • Check the FTP port number and the name of the iSphere library. The default values should be: Port 21 and ISPHERE.
  • Click "Transfer iSphere library" and enter the name of your host system and the user and password that shall be used to connect to the host.

  • Click the "OK" button to proceed.
  • On the following dialog, click "Start Transfer" to start the FTP transfer.

Doing it By Hand

  • Unzip biz.isphere.core_x.x.x.jar from the iSphere Update Site zip file.
  • Unzip file ISPHERE from directory /Server of biz.isphere.core_x.x.x.jar.
  • Create an empty save file in QGPL:

    CRTSAVF FILE(QGPL/ISPHERE) TEXT('iSphere')

  • Start a FTP session and transfer the local ISPHERE file into the ISPHERE save file on your host using binary mode.
  • Restore library ISPHERE from the save file QGPL/ISPHERE:

    RSTLIB SAVLIB(ISPHERE) DEV(*SAVF) SAVF(QGPL/ISPHERE) RSTLIB(ISPHERE)
