Install Plug-in for WDSCi 7.0

Using the iSphere Update Site

From the main menu, select "Help > Software Updates > Find and Install..." to enter the "Install/Update" dialog.

Select "Search for new features to install" and press the [Next] button:

Now click the [New Remote Site...] button to enter the URL of the iSphere update site http://isphere.sourceforge.net/eclipse/wdsci7.0/:

Then click the [Finish] button and follow the instructions given on the following pages.

Using a Local Update Directory

Go to the iSphere download page at SourceForge and download the zip file for your environment.

The follow these steps to create your local update site:

  • Unpack the zip file to a directory of your choice. This directory will be your local update site.
  • From the main menu, select "Help > Software Updates > Find and Install..." to enter the "Install/Update" dialog.
  • Press the [New Local Site...] button to add your directory to the list of available update sites.

Using a Local Update Archive

Go to the iSphere download page at SourceForge and download the zip file for your environment.

The follow these steps to create your local update site:

  • Download the zip file to a directory of your choice. This zip file will be your local update archive.
  • Consider removing the version number from the name of the zip file to keep the name stable for further updates.
  • From the main menu, select "Help > Software Updates > Find and Install..." to enter the "Install/Update" dialog.
  • Press the [New Archived Site...] button to add the zip file to the list of available update sites.
