Upgrade Instructions

Your preferences will be moved automatically to the new version.

Upgrading (except to iSphere v2.0.0)

When you install an upgrade to any version, except to v2.0.0, follow the Install Instructions. iSphere will automatically remove the plug-ins of the prior version. Upgrading to iSphere v2.0.0 is an exception--see below.

Upgrading to iSphere version v2.0.0

If you are upgrading to iSphere v2.0.0 you must manually remove the plug-ins for the prior version. Follow a link below for the manual uninstall instructions:

    Uninstall WDSC
    Uninstall RDP or RDi

After you have uninstalled the prior version plug-ins, follow the Install Instructions.

Starting with v2.0.0 the iSphere project is open source and this required changing the plug-in IDs from de.taskforce to biz.isphere. Therefore, for this one upgrade, you must manually remove the plug-ins for the prior version. The uninstall instructions can also be found on the iSphere Wiki pages.
